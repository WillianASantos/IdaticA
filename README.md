## **README.md: IdáticA - Sua Busca Inteligente por Temas de Aula**

### **O que é a IdáticA?**

A IdáticA é uma ferramenta simples e intuitiva que te ajuda a encontrar o tema perfeito para sua próxima aula. Basta digitar uma palavra-chave e pronto! A mágica da programação te apresenta uma lista de sugestões personalizadas e interessantes.

### **Como funciona?**

1. **Digite o tema:** Na barra de pesquisa, escreva qualquer palavra relacionada ao assunto que você quer abordar.
2. **Clique em "Pesquisar":** A IdáticA vai vasculhar sua base de dados e te mostrar todos os temas relacionados à sua pesquisa.
3. **Explore os resultados:** Cada resultado te dá um resumo do tema, os principais pensadores envolvidos e um link para você saber mais.

**Por trás da magia...**

A IdáticA usa um pouquinho de JavaScript para fazer toda a mágica acontecer. Quando você clica em "Pesquisar", o JavaScript:

* **Pega a sua palavra-chave:** E transforma ela toda em letras minúsculas para facilitar a busca.
* **Vasculha a base de dados:** Essa base de dados está guardada num arquivo JavaScript chamado `dados.js`.
* **Compara as palavras:** O JavaScript compara a sua palavra-chave com todas as palavras presentes nos títulos, descrições, pensadores e tags dos temas da base de dados.
* **Mostra os resultados:** Se encontrar alguma correspondência, ele cria uma listinha bem bonitinha com os resultados da pesquisa.

### **Tecnologias utilizadas:**

* **HTML:** A estrutura básica da página. É como a ossatura do nosso site.
* **CSS:** Deixa tudo bonitinho e organizado. É a roupa da nossa página.
* **JavaScript:** A parte mais inteligente. Faz a página interagir com você e te mostrar os resultados da pesquisa.

### **Para que serve?**

A IdáticA é perfeita para professores que querem inovar nas suas aulas e encontrar temas que realmente interessem os alunos. É também uma ótima ferramenta para quem está estudando por conta própria e precisa de inspiração para novos projetos.

**Observação:** Essa é uma versão bem simples da IdáticA. Com um pouco mais de trabalho, podemos adicionar novas funcionalidades, como:

* **Filtros:** Para refinar ainda mais os resultados da pesquisa.
* **Recomendações:** Para sugerir temas relacionados ao que você já pesquisou.
* **Integração com outras ferramentas:** Como plataformas de ensino a distância.
